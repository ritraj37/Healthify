# Healthify Server Setup Steps

## 1. Open Command Prompt
- Press Windows + R
- Type: cmd
- Press Enter

## 2. Navigate to Backend Folder
```
cd /d D:\Healthify\healthify-backend
```

## 3. Install Dependencies
```
npm install
```

## 4. Start Server
```
npm start
```
