# Healthify - Healthcare Website

A comprehensive healthcare website built with modern web technologies, featuring responsive design, AI assistance, and complete medical services.

## 🏥 Features

### Core Pages
- **Homepage** - Hero section with video background, services overview, doctor profiles, patient reviews
- **About Us** - Company story, mission, vision, values, and achievements
- **Services** - Comprehensive medical specialties and treatments
- **Doctors** - Professional profiles with ratings, specializations, and booking
- **AI Assistant** - Interactive health chatbot with intelligent responses
- **Appointment Booking** - Online appointment scheduling system
- **Emergency Services** - 24/7 emergency contact and information
- **Blog** - Health articles with category filtering and full article reading
- **Contact** - Multi-channel contact options with professional form

### User Authentication
- **Login System** - Secure patient login portal
- **Registration** - New patient registration with validation
- **Privacy Policy** - HIPAA-compliant privacy documentation
- **Terms of Service** - Legal terms and conditions

### Technical Features
- **Responsive Design** - Mobile-first approach with breakpoints for all devices
- **SEO Optimized** - Meta tags, sitemap, robots.txt, and structured data
- **Interactive Elements** - Smooth animations, hover effects, and transitions
- **AI Chatbot** - Intelligent health-focused responses without external APIs
- **Article Modal System** - Full article reading with detailed medical content
- **Form Validation** - Client-side validation for all forms
- **Accessibility** - WCAG compliant design and navigation

## 🚀 Technologies Used

- **HTML5** - Semantic markup and modern web standards
- **CSS3** - Advanced styling with Flexbox, Grid, and animations
- **JavaScript (ES6+)** - Interactive functionality and DOM manipulation
- **Font Awesome** - Professional medical and UI icons
- **Google Fonts** - Roboto font family for clean typography

## 🔧 Setup Instructions

1. **Clone or Download** the project files
2. **Open** `index.html` in a web browser
3. **For Development**: Use a local server (Live Server, XAMPP, etc.)
4. **For Production**: Upload to web hosting service

### Local Development
```bash
# Using Python (if installed)
python -m http.server 8000

# Using Node.js (if installed)
npx http-server

# Using PHP (if installed)
php -S localhost:8000
```
