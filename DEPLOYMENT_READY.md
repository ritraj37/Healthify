## ⚡ QUICK DEPLOYMENT (3 STEPS)

### Step 1: Update URLs (5 minutes)
```bash
# Windows
deploy-helper.bat

# Linux/Mac
chmod +x deploy-helper.sh
./deploy-helper.sh
```

### Step 2: Update Payment Keys (2 minutes)
Open `payment.html` and update:
```javascript
key: 'rzp_live_XXXXXXXXXX'  // Your live Razorpay key
```

### Step 3: Upload & Test (10 minutes)
- Upload all files to server
- Install SSL certificate
- Test all features
- Go live! 🎉
